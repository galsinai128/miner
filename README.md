# LUCKY MINER

This is the lucky miner implementation home assignment

## Installation

- Clone the repo
- cd miner
- npm start
- open browser on localhost:3000

## Whats implemented

- Fake backend with loading timeout of 500 miliseconds
- game implementation - like mockup
- ui with sass
- animations
- cashout
